## Installation

* Stardogの最新版をダウンロードして解凍
```
wget https://downloads.stardog.com/stardog/stardog-latest.zip
unzip stardog-latest.zip
```

* 以下のドキュメントに従って設定とStardogの起動を行う。
https://www.stardog.com/docs/#_setting_path
